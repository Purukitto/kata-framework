Stable v1.3.0 release — part of the coordinated Kata Framework v1.0.0 launch. `kata-react` was already published at 1.2.0 on npm, so this ships as a minor bump to avoid a hostile downgrade for existing consumers. There are no breaking changes; the public surface (KataProvider, useKata, useKataMultiplayer, TypewriterText, SceneTransition, TweenTarget/useTween, SaveManager/useSaveSlots, KataErrorBoundary, a11y hooks) is now frozen under semver. Future breaking changes require a new major version.
