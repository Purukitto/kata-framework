## Stability

All packages listed above are at **v1.0.0** and follow semver. Breaking changes require a major bump. See the [changelog](https://github.com/purukitto/kata-framework/blob/main/CHANGELOG.md) for release history.

## Finding things

- **Looking for a hook?** Start in `@kata-framework/react` — everything UI-side lives there.
- **Looking for a plugin?** They live in `@kata-framework/core/plugins/*` subpaths. The [plugins catalog](/kata/docs/guides/plugins-catalog) is usually faster than browsing the API tree.
- **Looking for types like `KSONFrame` or `KSONAction`?** They're re-exported from `@kata-framework/core`'s root. The [KSON protocol page](/kata/docs/reference/kson-protocol) is a friendlier entry point.

Use the search box (`/` shortcut) to find a symbol by name across all packages.
